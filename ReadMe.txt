# Loan Approval Prediction Tool

## Overview
This project is a loan approval prediction tool designed for Trublu Financial Services Limited. It utilizes statistical analysis to present loan data in useful graphical representations, aiding in decision-making processes related to loan approvals.

## Features
- **Data Analysis**: The tool analyzes various factors influencing loan approvals, including age, income, loan amounts, and loan intents.
- **Visualizations**: It generates insightful graphs that illustrate:
  - Average loan amounts by loan intent.
  - Approval rates segmented by home ownership status.
  - Summary statistics of key variables such as credit scores and income.

## Key Components
- **Data Manipulation**: Utilizes `pandas` for data manipulation and analysis.
- **Data Visualization**: Employs `matplotlib` and `seaborn` to create informative visualizations.
- **Statistical Summaries**: Provides detailed statistical summaries of the dataset.
